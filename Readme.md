##Visual Code Auto Save Config
Open File > Preferences > Settings and paste the code:
``
{
    "files.exclude": {
        "**/.git": true,
        "**/.DS_Store": true
    },
    // Control auto save
    "files.autoSave": "afterDelay"
}
